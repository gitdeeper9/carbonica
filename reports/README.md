# CARBONICA Reports Directory

## Directory Structure

- `daily/` - Daily reports (TXT format only)
- `weekly/` - Weekly summary reports
- `monthly/` - Monthly analysis reports
- `alerts/` - Alert notifications

## Report Formats

- Daily reports: Plain text (.txt)
- Weekly reports: Markdown (.md) + CSV
- Monthly reports: Markdown (.md) + JSON
- Alerts: Plain text (.txt)

## Naming Convention

- Daily: `YYYY-MM-DD_report.txt`
- Weekly: `YYYY-WW_summary.md`
- Monthly: `YYYY-MM_analysis.md`
- Alerts: `YYYY-MM-DD_HH-MM_alert.txt`
